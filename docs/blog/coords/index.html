grid 3：图形坐标系统

王诗翔

Monday, 9 August 2021

绘图时，我们需要坐标才能准确地对要绘制的对象进行定位。在 grid 包中，有多种绘图坐标系统，选择一个合适的加以利用可以帮助我们更好地绘制图形。

坐标系统不同的单位：

  1. native 单位：根据数据值。

  2. n pc单位：将整个（笛卡尔）坐标系缩放为0到1范围。

  3. 实际尺寸单位：包括英寸、厘米、毫米。

例如，下面的绘图同时用到了前两者：

 1ex_vp <- viewport(x = 0.5, y = 0.5, # npc 单位
 2                  just = c("center", "center"),
 3                  height = 0.8, width = 0.8,
 4                  xscale = c(0, 100), yscale = c(0, 10)) # 设定native单位
 5pushViewport(ex_vp)
 6grid.draw(rectGrob())
 7grid.draw(circleGrob(x = unit(20, "native"), y = unit(5, "native"), # 根据native单位绘图
 8                     r = 0.1, gp = gpar(fill = "lightblue")))
 9grid.draw(circleGrob(x = unit(85, "native"), y = unit(8, "native"),
10                     r = 0.1, gp = gpar(fill = "darkred")))
11popViewport()
