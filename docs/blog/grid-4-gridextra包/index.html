grid 4：gridExtra包

王诗翔

Monday, 9 August 2021

gridExtra包提供了grid系统的有用拓展，包括对grobs对象的操作方法和其他一些grobs对象。

grid.arrange()可以绘制多个对象：

1library(gridExtra)
2grid.arrange(lollipop, circleGrob(),
3             rectGrob(), lollipop, 
4             ncol = 2)

由于ggplot2也是基于grid系统的，所以我们可以使用该函数对ggplot对象组合排序：

1time_vs_shots <- ggplot(worldcup, aes(x = Time, y = Shots)) + 
2  geom_point()
3player_positions <- ggplot(worldcup, aes(x = Position)) + 
4  geom_bar()
5
6grid.arrange(time_vs_shots, player_positions, ncol = 2)

使用layout_matrix选项可以进行更自定义控制：

1grid.arrange(time_vs_shots, player_positions,
2             layout_matrix = matrix(c(1, 2, 2), ncol = 3))

包括图形填充：

1grid.arrange(time_vs_shots, player_positions,
2             layout_matrix = matrix(c(1, NA, NA, NA, 2, 2), 
3                                    byrow = TRUE, ncol = 3))

使用tableGrob()函数，我们可以在图中嵌入表格。下面是一个略微复杂的例子：

 1worldcup_table <- worldcup %>%
 2  filter(Team %in% c("Germany", "Spain", "Netherlands", "Uruguay")) %>%
 3  group_by(Team) %>%
 4  dplyr::summarize(`Average time` = round(mean(Time), 1),
 5                   `Average shots` = round(mean(Shots), 1)) %>%
 6  tableGrob()
 7`summarise()` ungrouping output (override with `.groups` argument)
 8
 9grid.draw(ggplotGrob(time_vs_shots)) # 先绘制了一个ggplot
10wc_table_vp <- viewport(x = 0.22, y = 0.85, 
11                        just = c("left", "top"),
12                        height = 0.1, width = 0.2) # 添加一个新的视图绘制表格
13pushViewport(wc_table_vp)
14# 绘制表格
15grid.draw(worldcup_table)
16popViewport()
