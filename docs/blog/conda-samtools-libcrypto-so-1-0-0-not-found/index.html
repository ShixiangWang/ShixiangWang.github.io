又是xx.so找不到的conda常见问题，但每一种解决办法都不同。

参考 https://github.com/merenlab/anvio/issues/1479 和 https://github.com/bioconda/bioconda-recipes/issues/13958。

使用下面命令可以解决：

conda install -c bioconda samtools=1.9 --force-reinstall
